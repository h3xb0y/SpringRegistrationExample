- Spring Boot
- Spring Data JPA
- Maven
- JSP
- HIBERNATE

via terminal:
```
mysql -u root -p
CREATE DATABASE db_demo;
CREATE USER 'h3x'@'localhost' IDENTIFIED BY 'pass';
GRANT ALL PRIVILEGES ON *.* TO 'h3x'@'localhost';
FLUSH PRIVILEGES;
USE db_demo;
INSERT INTO `role` VALUES (1,'ADMIN');
INSERT INTO `role` VALUES (2,'USER');
```
